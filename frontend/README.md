# Laxmi Electronics - Frontend

React frontend application for the Laxmi Electronics website.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

4. Preview production build:
   ```bash
   npm run preview
   ```

## Development

The development server runs on `http://localhost:5173` by default.

Make sure the backend server is running on port 3001 for full functionality.

## Project Structure

```
frontend/
├── src/
│   ├── components/    # Reusable React components
│   ├── pages/         # Page components
│   ├── styles/        # CSS styles
│   └── utils/         # Utility functions
├── public/            # Static assets
└── index.html         # HTML entry point
```

## Technologies

- React 18
- React Router DOM
- Vite
- Tailwind CSS
